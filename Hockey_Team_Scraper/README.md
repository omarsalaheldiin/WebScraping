# Hockey Teams Information Scraper

This project scrapes hockey teams information (team name, year, wins, losses, win%, goals for (GF), goals against (GA)) from the Scrape This Site website and saves it to a CSV file.

## How to Use

1. Run the Jupyter notebook `Hockey_Teams_Scraper.ipynb`.
2. The notebook fetches the teams data and saves it to `hockey_teams_data.csv`.
3. The data includes team name, year, wins, losses, win percentage, goals for, and goals against.

## Requirements

- `requests`
- `beautifulsoup4`
- `pandas`
